About Data

  • There are two sections in this lab. On section one, you will work on the provded data set using the rpovided technique. For the second part you should apply the techniques on News popularity data set.

    • Predicting and Evaluating the Popularity of Online News by He Ren and Quan Yang,

  • The URL for the data set description is: http://archive.ics.uci.edu/ml/datasets/Online+News+Popularity.
